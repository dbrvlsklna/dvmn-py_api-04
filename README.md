# Публикация изображений космоса в Telegram-канал

Набор скриптов, автоматизирующий загрузку и публикацию изображений космоса в Telegram-канал.


## Основные скрипты

### Скачивание фото запусков SpaceX (fetch_spacex_images.py)
Скрипт скачивает оригинальные фотографии произведенных запусков ракет через официальный API SpaceX и сохраняет их в указанную директорию.

### Скачивание астрономических изображений дня (APOD) (fetch_daily_astronomy_images.py)
Скрипт скачивает случайные астрономические изображения дня (APOD) через API NASA и сохраняет их в указанную директорию.

### Скачивание цветных изображений планеты (EPIC) (fetch_epic_images.py)
Скрипт скачивает цветные изображения планеты, получаемые телескопом Earth Polychromatic Imaging Camera (EPIC) через API NASA, и сохраняет их в указанную директорию.

### Публикация скачанных изображений в Telegram (tgbot.py)

Cкрипт автоматизирует публикацию изображений в Telegram-канал с помощью бота.

- Одиночная публикация: отправляет одно конкретное изображение из указанной папки.
- Случайный выбор: отправляет одно случайное изображение из директории, если конкретный файл не указан.
- Автопубликация: запускает бесконечный цикл, который перемешивает все картинки в папке и отправляет их по очереди с заданным интервалом (задержкой), по умолчанию 4 часа.

## Запуск

### Установка зависимостей

Для запуска вам понадобится Python третьей версии. 
Перед запуском установите зависимости:

```pip install -r requirements.txt```

## Настройка окружения

Для работы скриптов необходимо создать файл `.env` в корневой директории проекта и добавить в него настройки для доступа к API Telegram и NASA.

### Создание файла конфигурации
Создайте в папке с проектом текстовый файл с именем `.env` и скопируйте в него следующий шаблон:

```env
TG_TOKEN=ваш_токен_телеграм_бота
TG_CHAT_ID=@имя_вашего_канала_или_id_чата
NASA_TOKEN=ваш_api_ключ_nasa
```

#### Получение токена Telegram
1. **Токен бота**: Откройте Telegram, найдите бота [@BotFather](https://t.me) и отправьте команду `/newbot`. Следуйте инструкциям, чтобы создать бота и получить его токен.
2. **ID канала**: Создайте публичный Telegram-канал. Добавьте вашего созданного бота в этот канал в качестве **Администратора** с правом публикации сообщений.
3. В переменную `TG_CHAT_ID` задайте имя вашего канала в формате `@имя_канала` (например, `@space_photo_channel`)..

#### Получение ключа NASA
1. Перейдите на официальный сайт [NASA APIs](https://nasa.gov).
2. Заполните короткую форму регистрации (имя, фамилия, email).
3. Нажмите кнопку **Sign Up**. Ваш персональный API-ключ отобразится на экране и придет на указанную почту.
4. Скопируйте его и вставьте вместо `ваш_api_ключ_nasa`.


### Скачивание фото запусков SpaceX

Скачать фото последнего запуска:

```python fetch_spacex_images.py /path/to/my_images```

Скачать фото конкретного запуска по его ID:

```python fetch_spacex_images.py /path/to/my_images -id 5eb87d47ffd86e000604b38a```

### Скачивание астрономических изображений дня (APOD)

Для запуска срипта вам необходимо ввести путь директории, куда вы планируете сохранять изображения, а также указать количество скачиваемых изображений.

```python fetch_daily_astronomy_images.py /path/to/my_images 3```

### Скачивание цветных изображений планеты (EPIC)

Для запуска срипта вам необходимо ввести путь директории, куда вы планируете сохранять изображения иуказать количество запрашиваемых изображений.

```python fetch_epic_images.py /path/to/my_images```

### Публикация скачанных изображений в Telegram

Одиночная публикация случайной картинки:

```python tgbot.py /path/to/my_images```

Публикация конкретного изображения:

```python tgbot.py /path/to/my_images -i /path/to/my_images/spacex_0.jpg```

Бесконечная автопубликация с задержкой по умолчанию (4 часа/14400 секунд):

```python tgbot.py /path/to/my_images -a```

Бесконечная автопубликация с заданной пользователем задержкой:

```python tgbot.py /path/to/my_images -a -d 60```
